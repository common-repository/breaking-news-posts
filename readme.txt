== Description ==

The Breaking News Posts plugin allow you to display a slider with break news links right after your navigation.
Inside your dashboard you can control the:
Slider speed
Post type
Number of posts

In future updates we have plans to include a shortcode and some additional options related to styling.


Major features in Breaking News Posts include:

* Slider speed
* Post type
* Number of posts


== Installation ==

1- Upload the Breaking News Posts plugin to your blog, activate it.
2- Inside Dashboard > Appearance > Menu > Menu Name => primary

The menu that's using the Name 'primary' will be the one displaying the news. 

== Frequently Asked Questions ==
 
Q- How to select multiple post type
 
A- You can select Multiple post types by holding CTRL when clicking on it
